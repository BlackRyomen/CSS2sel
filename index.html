Selectori

Selectorii CSS sunt folosiți pentru a „găsi” (sau selecta) elementele HTML pe care doriți să le stilați. Putem împărți selectoarele CSS în cinci categorii:

  • Selectori simpli (selectați elemente în funcție de nume, id, clasă)
  • Selectori combinatoare (selectează elemente pe baza unei relații specifice dintre ele)
  • Selectori de pseudo-clase (selectează elemente pe baza unei anumite stări)
  • Selectori de pseudo-elemente (selectează și stilează o parte a unui element)
  • Selectori de atribute (selectați elemente pe baza unui atribut sau a valorii unui atribut)

Această pagină va explica selectorii elementari CSS. 

Selectorul de elemente CSS

Selectorul de elemente selectează elemente HTML pe baza numelui elementului. În exemplul de mai jos, toate elementele <p> vor fi centrate, cu o culoare roșie a textului. Exemplu 1

 Selectorul id CSS

Selectorul de id folosește atributul id al unui element HTML pentru a selecta un anumit element. Id-ul unui element este unic într-o pagină, astfel încât selectorul de id este folosit pentru a selecta un element unic. Pentru a selecta un element cu un anumit id, scrieți un caracter hash (#), urmat de id-ul elementului. Regula CSS de mai jos va fi aplicată elementului HTML cu id="para1": Exemplu2

 Selectorul de clasă CSS

Selectorul de clasă selectează elemente HTML cu un anumit atribut de clasă. Pentru a selecta elemente cu o anumită clasă, scrieți un punct (.), urmat de numele clasei. În acest exemplu, toate elementele HTML cu class="center" vor fi roșii și aliniate la centru: Exemplu3

De asemenea, puteți specifica că numai anumite elemente HTML ar trebui să fie afectate de o clasă. În acest exemplu, numai elementele <p> cu class="center" vor fi roșii și aliniate la centru: Exemplu4

Elementele HTML se pot referi și la mai multe clase. În acest exemplu, elementul <p> va fi stilat conform class="center" și class="large": Exemplu5

 Selectorul universal CSS

Selectorul universal (*) selectează toate elementele HTML de pe pagină. Regula CSS de mai jos va afecta fiecare element HTML din pagină: Exemplu6

 Selectorul de grupare CSS

Selectorul de grupare selectează toate elementele HTML cu aceleași definiții de stil. Uitați-vă la următorul cod CSS (elementele h1, h2 și p au aceleași definiții de stil): Exemplu7
